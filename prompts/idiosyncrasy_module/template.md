# Idiosyncrasy Module Prompt Template

Use this prompt to generate the idiosyncrasy module for a character spec after elaboration and before extraction.

## Instructions

- Produce a schema-like module describing idiosyncratic behaviors, voice, and constraints.
- Keep output concise and compatible with downstream prompt insertion.
- Do not restate the full base spec; focus on deltas and unique behaviors.

## Inputs

- Elaborated base specification
- Any author notes relevant to idiosyncrasies

## Output

Return only the idiosyncrasy module content.
